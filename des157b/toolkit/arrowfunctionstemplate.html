loop

logs "thingToPrint" 'times' times to the console. I came up with it after hours of painstaking brainstorming.

Named Function

            function loop(thingToPrint, times){
                for (let i = 0; i < times; i++) {
                    console.log(thingToPrint);
                }
            }

Function Expression

            const functExp = function(thingToPrint, times){
                for (let i = 0; i < times; i++) {
                    console.log(thingToPrint);
                }
            }

Arrow Function

            const arrFunc = (thingToPrint, times) => {
                for (let i = 0; i < times; i++) {
                    console.log(thingToPrint);
                }
            }
