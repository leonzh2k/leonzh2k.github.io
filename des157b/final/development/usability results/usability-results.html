Usability Results

Observation 1

Most of the struggles occurred in the create page. One user said it wasn't obvious to tell which song they selected. Another said they didn't know what to do after pressing submit, since it didn't give any visual cues on where to view their viz. Also, they didn't know if the submit button actually worked, because there were no visual cues. As a result, there were multiple of the same song in the gallery.

Observation 2

A common thing I heard was that it wasn't obvious how to do some things. One user said they didn't really know how to play/pause the viz and got lucky by trying clicking. Another said they would like to have a tutorial on how to to everything.

Observation 3

Finally, users said they would like more diversity in the visualizations, since I only have one viz type implemented right now. One user said a blob would be cool, and another user said they would like to see more irregular shapes.

Feedback Implementation

I will definitely add some more visual cues that make it obvious how to accomplish everything, such as a play/pause icon to click. Also, I'll add a tutorial in the create viz section. I will also add more interesting visualizations.
